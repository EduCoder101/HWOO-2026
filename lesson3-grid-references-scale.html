Lesson 3 · Grid References & Scale

Map Skills: Grid References and Scale
Lesson at a Glance
  • Teach four-figure grid references using the Sydney atlas map ('along the corridor, up the stairs')
  • Show real-world examples of maps that use grid references (bushwalking, emergency services, topographic)
  • Guided practice using grid references on a world map
  • Split the class in half: one stream works with a world map, the other with an Australia map. Pairs use the atlas to find specific locations, then plot them on blank A3 maps using grid references
  • Students write inquiry questions about the places they discover on post-it notes for the inquiry wall
  • Completed maps go up as classroom displays for the rest of the unit
  • Teach scale bars: what they are and how to use them to convert map distance to real distance
  • Estimation challenge: estimate distance first, then measure with scale bar, then verify on Google Maps
  • Round-the-world travel plan: students given five destinations, choose their route order, record grid references and distances for each stop

Learning Intention

Use grid references to locate places on a map and use scale to estimate real-world distance.

Success Criteria

  • Use a four-figure grid reference to locate a place on a map
  • Give the grid reference for a named location
  • Use a scale bar to estimate the distance between two points
  • Explain why maps use grid references and scale

Lesson Structure

1

Grid References

25 min · Whole class then guided practice

Explicit teaching of grid references

Introduce the grid reference system using the atlas map of Sydney. Establish the convention: the horizontal axis (eastings) is read first, then the vertical axis (northings). Teach the mnemonic 'along the corridor, up the stairs' and work through several examples as a class, moving from teacher-led to student-called answers. Students practise giving and finding four-figure grid references on the Sydney atlas map.

Where else do grid references appear?

Briefly show students 2–3 examples of grid references in real-world contexts: a bushwalking map, an emergency services map, and a topographic map extract. The point is not to analyse these maps in detail but to establish that grid references are a universal navigation tool, not just a classroom exercise. If your school's area has a published bushwalking or topographic map, use it – the local connection is more powerful than a generic example.

Guided practice on a world map

Shift to the world map in the atlas. Work through 3–4 grid references as a class to establish that the same system operates at any scale. This is a bridge to the matching exercise that follows and prepares students for coordinate work in Lesson 9.

Support
Provide a grid reference cue sheet with the mnemonic and a worked example that stays on the desk throughout the lesson. This is the reference point students return to whenever they lose the axis order. Printable cue sheet below.

2

Place Matching Exercise

25 min · Individual or pairs · Two streams

The class splits into two streams. Half work with a world map (A3 printout, country borders and names only – no cities, landmarks or other features marked), half with a map of Australia (A3 printout, state borders and names only). Each stream receives a set of 8 place cards, each containing a photograph, a short description, and the place name.

Students must use the atlas to find each place, then locate the corresponding position on their blank A3 map using the grid reference system. They record the grid reference, then arrange the card around the A3 map and draw a line connecting it to the correct location. The atlas is the navigation tool; the A3 map is the display.

Post-it notes and pens should be available at each table. As students read the place cards and discuss with their partner, they write down any questions about the places they are finding ('What is palm oil?', 'Why is Venice flooding?', 'How big is the Murray-Darling?'). These go onto an inquiry wall – either the question wall from Lesson 1 if it is still displayed, or a fresh sheet. These questions are not answered now. They sit and wait until the relevant content arrives later in the unit.

Fast Finishers
Pairs who complete their set early choose additional places to add to the map – either places of personal significance (somewhere they have travelled, where a family member lives) or places they find interesting in the atlas. They create their own mini place card (name, grid reference, one sentence about why they chose it) and add it to the display.
Support
Pre-highlight the approximate region on the A3 map where each place is located (e.g. a light circle around South America for Machu Picchu) so students are searching within a bounded area rather than scanning the entire map. Pair supported students with a confident atlas navigator.

Close: Pin or display both annotated maps on the classroom wall. These become reference displays for the remainder of the unit. Brief whole-class share: each stream nominates one place that surprised them and one question from the post-it notes that they most want answered.

3

Scale

20 min · Whole class then individual

Explicit teaching of scale bars

Introduce scale bars using the atlas. Establish the principle: a map is a scaled-down representation of the real world, and the scale bar tells you how to convert map distance to real distance. Demonstrate with one worked example on the atlas – measure a distance with a ruler, then use the scale bar to convert to kilometres.

Estimation challenge

Students work through 3–4 distance pairs at different scales (local, national, global). For each pair the sequence is:

  • Estimate the distance first (gut feel, recorded in books)
  • Measure on the atlas using the scale bar
  • Verify using the Google Maps distance tool (teacher-operated, projected)

After each round, discuss: how close were the estimates? What makes distance hard to estimate on a map? The estimation step is the most important part – it builds spatial intuition, not just procedural scale bar use.

4

Travel Plan

20 min · Individual

Students plan a round-the-world trip from Sydney. Each pair or individual receives one of five pre-made travel sets (see below), each containing five destinations drawn from the place cards used in Stage 2. Distributing different sets means pairs cannot simply copy each other's routes, and the closing discussion produces a wider range of comparisons. Students decide the order of their route, then for each stop they record: the grid reference, the estimated distance from the previous stop, and the scale-bar-calculated distance. The final entry returns to Sydney.

The teacher may provide a proforma for this task or have students work in their books. Printable proforma below.

How to Read a Grid Reference

Along the corridor, up the stairs
Step 1: Read the number along the bottom of the map (the easting). This is your first two digits.
Step 2: Read the number up the side of the map (the northing). This is your second two digits.
Step 3: Put them together. Easting first, northing second.
Example: To find grid reference 3847
→ Go along to line 38
→ Go up to line 47
→ The place is in the square above and to the right of where the lines cross.

Reference

Teaching Notes

▾

Grid reference axis order – address explicitly

Students will confuse eastings and northings. The mnemonic 'along the corridor, up the stairs' anchors the correct order (horizontal first, then vertical), but expect to reinforce this multiple times during Stage 1. When students give a grid reference incorrectly (northing first), do not simply correct it – show them where the wrong reference points to on the map. Seeing the error land in the wrong place is more instructive than being told the order.

Scale bar misconception – 'bigger map means bigger place'

Some students will assume that a place which takes up more space on a map is physically larger. The shift between atlas pages (Sydney map vs world map) is the moment this surfaces. When introducing scale, make the comparison explicit: Sydney takes up a full page in the atlas and the whole of Australia takes up a full page – are they the same size? Use this to establish that scale determines how much of the real world fits on the page, not how big the place is.

Inquiry wall – harvest, don't answer

The place cards will generate questions about content that arrives later in the unit. The inquiry wall captures these without derailing the lesson. If students ask directly, acknowledge the question, confirm it's a good one, and point them to the wall. Some of these questions will be answered directly by Lessons 13–15; others will surface during the World Heritage investigation. The wall becomes more powerful the longer it accumulates.

Atlas Work

▾

When: Stages 1, 2, 3 and 4 – the atlas is the primary tool throughout this lesson.

Task: Stage 1: locate places on the Sydney atlas map using four-figure grid references, then repeat on the world map. Stage 2: use the atlas to find each place from the place cards and determine its grid reference on the blank A3 map. Stage 3: use the scale bar on three different atlas maps (Sydney, Australia, world) to measure distances. Stage 4: use the world map to plan a travel route with grid references and distances.

Page reference: Confirm against school atlas edition before the lesson and insert page numbers for: Sydney map with grid, Australia map with grid, world map with grid.

Observational Assessment

▾

What to look for

  • Stage 1 – grid reference accuracy: Are students reading eastings before northings consistently? Can they give a grid reference for a named place (the reverse operation), or only find a place from a given reference?
  • Stage 2 – matching exercise: Are students using the atlas and grid system to locate places, or scanning the A3 map visually? Listen to pair conversation – are they articulating the process ('go along to 34, then up to 57') or guessing? Note also the quality of questions going onto the inquiry wall.
  • Stage 3 – estimation and scale: How far off are the initial estimates? Can students use a ruler and scale bar independently?
  • Stage 4 – travel plan: Can students combine both skills (grid reference + scale) in an applied context? This is the real test of whether the lesson has landed.

How to capture

Phase 1 Observation Checklist. Columns: Uses grid references · Uses scale bar. Code during Stages 2 and 4 – these are the independent application moments.

What it tells you

Students who cannot reliably give or find a four-figure grid reference by the end of Stage 2 will need a brief check-in before Lesson 4, where grid references are used in the annotated local map work. Students who struggle with scale bar use should be flagged – the skill is reinforced through Lessons 6–14, so it is not essential that every student is fluent by the end of this lesson, but those who cannot attempt the process at all will need targeted support.
