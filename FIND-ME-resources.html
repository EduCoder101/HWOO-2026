Maps & Atlases
Paper atlases (class set)
One per student. Must include: Sydney region page, NSW/Australia page, world map page with grid reference axes. Check the school library or geography department.
L1L2L3L4L5+
Cammeraygal Country map
Indigenous map of the local area showing Cammeraygal Country. Used as a primary source alongside modern maps, not as a supplement. Check school curriculum resources or contact your Aboriginal Education team.
L1L4
Six-map set of Roseville area
Six A4 maps all showing the same area: satellite image, street map, train network map, topographic map, historical map, and Cammeraygal Country map. One set per pair. You will need to source and print each individually.
L1
Satellite and street maps: screenshot from Google Maps. Train network: Transport NSW website. Topographic: NSW Spatial Services. Historical: search State Library of NSW digital collections.
Mercator and Gall-Peters projection images
Two world map projections displayed side by side. Projected on IWB or printed A3. The visual contrast between the two drives the discussion.
L1
Search "Mercator vs Gall-Peters" in Google Images. Many free comparison images available.
Printed local street map (Roseville/North Shore)
A4 or A3 street map with landmarks, compass rose, and key. Needs minimum 10 identifiable landmarks and at least 8 map symbols visible at print scale.
L2
Screenshot from Google Maps or OpenStreetMap. Add compass rose and key if not present.
Blank A3 world map (with country names and grid axes)
One per pair. Must have grid reference axes along the edges (letters on one axis, numbers on the other). Country names should be visible but the map should otherwise be uncluttered.
L3
Search "blank world map with grid references A3 printable" or create one from an outline map.
Blank A3 Australia map (with state names and grid axes)
One per pair. Grid reference axes along edges, state/territory names visible. Used alongside the world map in the split-class activity.
L3
Outline map of Roseville/North Shore (for annotation)
A3 or larger blank map showing coastline, major roads, waterways, and grid reference axes but no feature labels. One per group. Used as the foundation for the annotated class map in Stage 3.
L4
Trace or simplify a Google Maps satellite screenshot. Add grid axes.
Photographs & Images
Provocation photograph of Roseville
High-resolution photograph of the school's immediate surroundings showing a mix of built and natural features. Projected on IWB or printed A3. This stays displayed through multiple stages.
L1
Walk around the school and take a photo that shows both natural vegetation and built features (buildings, roads, signage) in one frame.
Place card photographs (16 images)
Photographs for the place cards used in L3. Need images for 8 world locations (e.g. Machu Picchu, Great Barrier Reef, Sydney, etc.) and 8 Australian locations (e.g. Kakadu, Broken Hill, Hobart, Coober Pedy, etc.). Each image should be recognisable and printable at playing-card size.
L3
Search for Creative Commons images. Wikimedia Commons is a good source for landmark photos.
Digital Resources (load before class)
Google Maps – Roseville College (street view, satellite, terrain)
Pre-load on the IWB computer before class. Start in street view at the school entrance. Test that satellite and terrain layers switch smoothly. Needs reliable internet.

Compass rose drawing video
YouTube instructional video (youtube.com/watch?v=swXwvbN_TC8). Students draw along. Test playback and check school firewall before the lesson.
L2
HTML symbol challenge (digital interactive)
Digital map symbol identification activity loaded on tablets or laptops (one per pair). Requires WiFi. If unavailable, use the printed version instead.

Physical Equipment
Real compasses (class set)
One per pair minimum. Used for classroom compass work and the outdoor orienteering challenge. Check the science or geography department.
L2
Butcher's paper for Question Wall
One large sheet, headed "What do we want to know about places?" Displayed on the classroom wall for the entire unit. Position where students can see it from their seats.
L1L3
Large paper sheets (A2 or butcher's paper) for annotation
One per group for the annotated feature map. Students mark physical and human features using colour coding.
L4
Local landmarks word bank
A4 sheet with 15–20 local features with simple illustrations or icons. For support students during the feature investigation. You will need to create this with Roseville-specific landmarks.
L4
Setup Tasks (do before the lesson)
Orienteering course (L2)
Place orienteering instruction cards at locations around the school grounds. Mark a 10m distance with chalk or tape for stride calibration. Allow 15–20 minutes setup before the lesson.
L2
School grounds map (L2, optional)
Simple A4 overhead view of the school grounds for the orienteering scaffold. Only given to students who request it. Create a basic sketch if one doesn't exist.
